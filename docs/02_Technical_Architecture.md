# Technical Architecture

## Selected stack

- Flutter
- Riverpod
- GoRouter
- Material 3
- Supabase and PostgreSQL
- Supabase Storage
- Firebase Cloud Messaging
- Drift for local offline storage
- GitHub Actions

## Layers

- Presentation
- Application
- Domain
- Infrastructure

The starter implements the presentation shell first. Backend and offline
dependencies will be introduced incrementally to keep the first build reliable.
