# Auth & Family Setup v1.1.0

הגרסה כוללת את כל הזרימות והממשקים של Authentication ומשפחות.

## מצב החיבור לענן

בגרסה זו קיימת שכבת State מקומית לצורך UI, בדיקות ובנייה יציבה.
בגרסה הבאה של התשתית נחבר את אותם Controllers ל-Supabase Auth ול-PostgreSQL,
בלי לשנות את המסכים או את הניווט.
